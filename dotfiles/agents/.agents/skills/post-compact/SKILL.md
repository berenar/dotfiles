Re-orient after a compaction. Give the user a short, scannable summary of the current state of the work.

Report:
- **Goal** — what we're ultimately trying to achieve
- **Where we are** — what's been done so far
- **Next** — the immediate next step

Keep it to a few lines. Surface only what's needed to continue confidently. If anything is unclear after the compaction, say so instead of guessing.
